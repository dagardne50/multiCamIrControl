# multiCamIrControl
IR remote from Arduino to any brand of camera

Download and place this library into your arduino library folder.
The electronic part is simple, connect a IR led and 100Ohm resistor ( current limitation ) from 1 pin of the arduino.
The code allows you to set this pin.

This is an update from
